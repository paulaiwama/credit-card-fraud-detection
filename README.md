# Machine Learning e Cibersegurança Financeira

## Detecção de Fraudes em Cartões de Crédito

Este repositório contém os notebooks desenvolvidos para o Trabalho de Graduação apresentado à Faculdade de Tecnologia de Ourinhos (Fatec Ourinhos).

## Objetivo

Comparar o desempenho dos modelos:

- CCNN
- CFTNet
- SICRNN

utilizando os datasets:

- Kaggle Credit Card Fraud Detection
- Sparkov
